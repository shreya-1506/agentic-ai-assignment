Task 3: PDF Source Linking for Extracted Data

Objective: Improve traceability of extracted financial data.

Requirements:

Every extracted financial number should include a source reference.
Clicking the reference opens the source PDF.
Automatically navigate to the exact page (and ideally highlight the extracted value).
Support all extracted metrics used throughout the application.

Acceptance Criteria:

Every financial value is traceable back to its source.
Users can validate extracted information with one click.
